# Project: Second Life Marketplace UI

## Overview
Design and implement a modern, responsive UI for a marketplace platform called **Second Life**, where users can **buy and rent second-hand items**.

⚠️ IMPORTANT:
- FRONTEND ONLY (no backend implementation, no proxy setup)
- Assume APIs already exist
- Focus on UI/UX, component structure, and interaction flows

---

## 🎨 Design System

### Style
- Clean, modern, slightly cute
- Rounded corners everywhere
- Use **pastel green + white** as primary colors
- Use **soft gradients** across components
- Background: white with subtle illustrated icons (bags, furniture, etc.)
- Soft shadows and glassmorphism where appropriate

### UX Requirements
- Fully responsive (Desktop + Mobile)
- Smooth animations (hover, modal, transitions)
- Skeleton loading states
- Empty states for lists

---

## 👥 Roles

- Guest (not logged in)
- User (logged in)

---

## 🔐 Authentication UI

### Login
- Email + password
- Toggle show/hide password
- JWT stored in cookies (handled externally)

### Register
- Email + password
- After submit → show:
  "Please verify your email. Check spam if not found. Link valid for 24 hours."

### Edge Case
- If login returns empty token:
  → Show email verification notice again
