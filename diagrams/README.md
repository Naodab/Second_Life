## Diagram Files

To view or edit the diagrams, open the corresponding `.drawio` files using **draw.io Desktop** or the online editor at **https://app.diagrams.net/**.

### Activity Diagrams

```text
activities/
└── ActivityDiagrams.drawio
```

### Database ER Diagrams

```text
databases/
└── erd.drawio
```

Similar for other diagrams

The `.jpg` files in the `databases/` directory are exported images for quick reference only. If you need to modify a diagram, please edit the original `.drawio` file.
